PBSawatea Changes
=================
Authors : RH  = Rowan Haigh
          AME = Andrew Edwards
          PJS = Paul Starr

0.13  2014-09-26
  - Revised R code and Sweave code to deal with more than one commercial gear type. (RH 9/26)
  - Added vignette to briefly describe the package and past DFO assessments using Awatea. (AME/RH 8/26)
  - Modified function `runSweaveMCMC' to include a new argument `skip.last.year' for compatibility with older 2010 POP 5ABC assessment. (RH 8/26)
  - Modified Sweave master documents `run-master.Snw' and `run-masterMCMC.Snw' for compatibility with older 2010 POP 5ABC assessment. (AME 8/25)
  - Added 2010 POP 5ABC input files. (AME 8/25)

0.12  2013-11-12
  - SGR and ROL assessments finished. Cleaned up code and Rd files for incorporation into R-package. (RH 11/12)
  - Sweave MCMC can now take in tex files of MPD results (that user prepares by extracting pieces from run-master.Snw build). (RH 11/4)
  - Added a few more flags to determine if the Sweave file is meant to be a working paper or a ResDoc (`@rmresdoc` used for both). (RH 10/1)
  - Revised the Sweave MCMC to allow the inclusion of a discussion file specific to a single run (e.g., `resultsMCMC-run22.tex`). (RH 9/30)
  - More updates and debugging, mostly for MCMC stuff. (RH 9/27)
  - Debugged problems in Sweave MCMC. (RH 9/20)
  - Revised `run-MasterSweave.Snw` and `runSweaveMCMC.r`. (RH 9/16)
  - Revised `run-Master.Snw` and various functions. (RH 9/12)
  - Revised `runSweave` and various other functions to work with 2 sexes; improved some of the output figures. (RH 9/6)
  - Revised extensivley various functions to deal flexibly with one-sex models and optional CPUE series. (RH 9/5)
	Recoding occurred in `runADMB.r`, `runSweave.r`, `PBSscape.r`, and `run-Master.tex`.

0.11  2013-02-20
  - Updated just for compatibility with R-3.0.0. (RH 4/19)
  - Revised functions `importProjRec` and `plt.mcmcGraphs`; revised `run-masterMCMC.Snw`. (AME 2/20)

0.10  2012-10-16  Rowan
  - Revised function 'msyCalc' to report the convergence numbers. (AME 10/16)
  - Updated function 'readAD' to handle inactive data identified by "###" as opposed to comments identified by "#". (RH 10/16)
  - Updated functions 'pltmpdGraphs' and 'plt.mcmcGraphs' (e.g., lower panel of pairs plot reports the correlation coefficient). (AME 10/15)
  - Updated function 'runSweaveMCMC' and 'run-masterMCMC.Snw' to accommodate variable numbers of pairs plots. (RH 9/10)
  - Tweaked margins for figures in function 'plotIndexNotLattice'; updated Rd files. (RH 9/6)
  - Updated functions in R files 'runSweave.r', 'runSweaveMCMC.r', and 'PBSscape.r';
	separated menu functions from 'PBSscape.r' to 'menuFuns.r'. (RH 8/21)
  - Updated Sweave files 'run-master.Snw' and 'run-masterMCMC.Snw'. (8/21)
  - Updated functions 'plt.mcmcGraphs' and 'plotSnail' to work more generally. (AME 8/17)
  - Cleaned and debugged 'run-masterMCMC.Snw' further; still not finished for fixed M & H options. (AME 8/17)
  - Updated 'panel.curve' in  function 'plotDensPOPparsPrior'. (AME 8/16)
  - Re-wrote the function that selects and generates prior distributions using 'switch'. (AME 8/15)
  - Fixed bug in function 'plotVBcatch'. (AME 8/15)
  - Debugged 'run-masterMCMC.Snw' up to prior functions section (AME will take it from there). (RH 8/14)
  - Resolved a minor bug in the function 'findTarget' which crashed when number of projection years were too short to calculate generation-time reference criteria. (8/14)
  - Added new function 'importEva' to import the Hessian eiganvalues from an Awatea file, usually called 'Awatea.eva'. (RH 8/10)
  - Updated 'plotBars' to determine start year from results file. (RH 8/10)
  - Added new function 'importLik' to import an Awatea likelihood file, usually called 'likelihood.dat'. (RH 8/10)
  - Added 'input' directory for the Awatea input files used; format = spp-area-run.txt. (RH 8/2)
  - Modified 'runADMB' to handle situations when 'Awatea.cor' and 'Awatea.std' are not available ( when -nohess is specified in MPD call). (RH 8/2)
  - Added new function 'plotBars' to illustrate how an age composition compares with an equilibrium age structure. (RH 8/1)
  - Modified 'importRes' to import virgin vulnerable and spawning biomass values. (RH 7/31)
  - Added new function 'importCor' to import the Awatea correlation file. (RH 7/30)
  - Added new functions 'importPar' and 'importStd'  to import all and estimated parameters, respectively. (RH 7/27)
  - Updated AWATEAdata class (not thoroughly tested yet). (RH 7/27)
  - Updated 'runSweave' to handle surveys with missing age composition data. (RH 7/26)
  - Updated 'runADMB' to find the system exectuable file 'Awatea.exe' using the 'awateaPath' argument. (RH 7/25)
  - Updated the 'runMPD' wrapper for 'runSweave'. (RH 7/25)
  - Updated 'runSweave' and 'run-Master.Snw' to be more flexible wrt the number of surveys. (7/19)
  - Updated 'runADMB', 'runSweave', 'runSweaveMCMC' to deal with POP in WCVI and WCHG. (RH 7/18)
  - Implemented Brian Ripley's suggestion to remove NOTES about 'no visible binding for global variable xxx':
	if(getRversion() >= "2.15.1") globalVariables(names=c(...), package="PBSawatea") (RH 7/17)
